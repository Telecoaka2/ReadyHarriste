# Interface web de l'église Harriste de France

L'Église harriste ou Église du Christ - Mission Harris est une Église indépendante de type prophétique 
qui a foi en Jésus Christ, surtout implantée en Côte d'Ivoire. Elle a été fondée au début du XXe siècle 
par William Wade Harris Wury. L'Église harriste fait partie du Conseil œcuménique des Églises (COE) depuis 1998.
